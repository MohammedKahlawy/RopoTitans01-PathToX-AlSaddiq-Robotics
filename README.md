# AlSiddiq Robotics 2026: Path to X Competition

## Project Overview
This repository documents our team's participation in the AlSiddiq Robotics 2026: Path to X Competition. The project focuses on designing and building an autonomous line follower robot equipped with a robotic arm capable of interacting with objects along the track. The repository includes hardware components, mechanical design, embedded software, documentation, and media that illustrate the full development process from concept to final competition day.

## Work Flow
1. **Competition Analysis**  
   - Studied the competition requirements, rules, and objectives.
   - Brainstormed possible approaches and robot configurations.

2. **Hardware Selection**  
   - Chose suitable motors, sensors, controllers, and mechanical components.
   - Verified compatibility and availability of parts.

3. **Mechanical Design**  
   - Designed the robot chassis and robotic arm structure using CAD software.
   - Simulated movements and stress points to optimize durability.

4. **Embedded Software Development**  
   - Implemented line following algorithms and robotic arm control.
   - Programmed sensor data processing and motor actuation.
   - Conducted iterative testing and debugging.

5. **Testing & Optimization**  
   - Performed multiple runs to identify performance bottlenecks.
   - Adjusted parameters, sensor thresholds, and mechanical alignments

6. **Final Preparation**  
   - Integrated all components and ensured reliability.

## Tools & Software Used
- **Arduino IDE v1.8.19** – Programming ESP32 and motor controllers
- **SolidWorks 2022** – Mechanical design and assembly
- **ESP32 Board Libraries v2.1.0** – For motor and sensor interfacing
- **kicad 9- used to design schematic diagram and PCB loyout

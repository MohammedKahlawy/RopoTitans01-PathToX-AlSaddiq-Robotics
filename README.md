AlSiddiq Robotics: Path to X Competition

This repository documents our team's work for the AlSiddiq Robotics 2026 Path to X Competition. The project focuses on designing and building 
an autonomous line follower robot equipped with a robotic arm capable of interacting with objects along the path. The repositoryincludes 
hardware components, mechanical designs, software development, documentation, and media that illustrate the full development process 
from concept to competition day.
